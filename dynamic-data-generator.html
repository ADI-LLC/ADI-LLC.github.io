Dynamic Data Generator™

The Dynamic Data Generator™ is an ExactData, LLC product sold exclusively by ExactData.

The Dynamic Data Generator™ automates the generation of large sets of fully synthetic (fake) test data that meet the consistency and realism requirements which software performance testing demands.


These data sets are Large Scale, Fully Synthetic, Realistic, and Engineered.

 

Characteristics & features include:

  • A wide variety of data types
  • Internally consistent data
  • Controlled distributions of random variables
  • Longitudinal data (consistent across time)
  • Pattern Injection
  • Consistency across disparate data stores
  • Unstructured data (ex. Doctors notes)
